Welcome to the repository for my personal portfolio website! This project serves as a central hub for my professional journey as an aspiring cybersecurity professional and SOC analyst, while also sharing a bit about my life as a part-time classical piano teacher and endurance cyclist.

Live Site: [https://Albi2612.github.io/](https://Albi2612.github.io/)

 🚀 About the Project

This portfolio is a single-page application built to showcase my skills, projects, and background. It highlights my technical foundation in HTML/CSS, JavaScript, and Python, alongside my specific focus on cybersecurity. 

The site is engineered for speed and a smooth developer experience, utilizing modern frontend tooling and deployed directly via GitHub Pages.

 🛠️ Built With

This project leverages a modern web development stack:

React - UI Library
TypeScript - For robust, statically typed code
Vite - Next-generation frontend tooling and bundler
GitHub Pages - Hosting and deployment
